## Migrate to Vuejs 3

```zsh
$ vue add vue-next
```

## Upgrade to latest npm packages

- Need version lense VS extension
- Go to package.json
- Upgrade all packages

```zsh
$ rm -rf node_modules
```

## Project setup

```zsh
$ npm cache clean -f
$ npm install
```

### Compiles and hot-reloads for development

```zsh
$ npm run serve
```

### Compiles and minifies for production

```zsh
$ npm run build
```
