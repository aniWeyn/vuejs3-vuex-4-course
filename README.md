# vue3-course

## Migrate to Vuejs 3

```
vue add vue-next
```

## Upgrade to latest npm packages

- Need version lense VS extension
- Go to package.json
- Upgrade all packages

```
rm -rf node_modules
```

## Project setup

```
npm cache clean -f
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
